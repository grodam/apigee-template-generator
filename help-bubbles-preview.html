Target Servers
Description
Les Target Servers définissent les endpoints backend pour chaque environnement. Ils permettent de découpler la configuration du proxy de l'URL réelle du backend.
Exemple
Host: services-test.sveafinans.no Path: /DebtCollection Port: 443
04
Inline Expandable Card
Qu'est-ce qu'un API Product?

Un API Product dans Apigee est un bundle d'APIs qui peut être consommé par les développeurs. Il définit les quotas, les scopes OAuth, et les politiques d'accès.

Format
org.team.api.v1
Environnement
dev, uat, prod

Qu'est-ce que le Rate Limiting?
Rate Limiting

Le Rate Limiting permet de contrôler le nombre de requêtes qu'un client peut effectuer dans un intervalle de temps donné.

  • Protection contre les abus
  • Garantie de disponibilité
  • Monétisation des APIs
